FAFU数据科学研讨班

2018-2019

本课程将从数据出发，以问题为导向，在解决问题中学习数据挖掘、机器学习等数据科学相关方法。以 Python 中的数据科学工具为开端，将重点介绍 NumPy、Pandas、Matplotlab/Seaborn、Scikit-learn 等工具，再逐渐深入到和数据科学相关的线性模型、决策树、神经网络、无监督学习等知识。

在课程结束时，预计同学将对数据科学与机器学习有很大的了解，能够灵活运用学习算法解决实际问题。

负责人：李俊 周建飞 陈云
时间：
地点：宝铃楼201

先决条件

  • 了解 Python 的基础语法
  • 熟悉基本的线性代数、微积分、概率论
